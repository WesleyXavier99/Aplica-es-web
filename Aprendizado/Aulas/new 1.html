Conhecendo o CSS

VocÊ conhece as formas de uso do CSS ? Já ouviu falar de seletores ?

Formas de implementação do css

  • CSS inline: aplicada a cada tag dentro do codigo HTML. É necessário inserir na tag 'style=': "parâmetro: valor"
  • CSS interno: aplicado no "" dentro da tag style. Permite a estilização de todas as ocorrências do seletor indicado na pagina HTML;
  • CSS externo: link para um arquivo externo que contém informações de estilos. Normalmente usado para layouts e google fonts

Seletores

É possível através da escolha dos setores definir onde os estilos criados serão aplicados:

  • seletor tag: ao usar a tag HTML como seletor, definimos que o estilo criado será aplicado em todas as ocorrencias da tag na página;
  • seletor classe: permite a criação de um conjunto de estilos que pode ser aplicada a tas específicas. Precisa do "." antes do nome no "
